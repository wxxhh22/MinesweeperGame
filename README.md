# MinesweeperGame
扫雷游戏带技能版
这是一款基于 Java Swing 开发的经典扫雷复刻游戏，界面清新简洁、操作流畅，在保留传统扫雷核心玩法的基础上，新增了五大趣味技能系统，让游戏更具策略性与可玩性。
玩家需要在网格中通过逻辑推理避开所有地雷，安全翻开所有非雷格子，并正确标记所有地雷即可通关。游戏内置计时、剩余地雷计数、重新开始、难度选择等完整功能，搭配点击、插旗、爆炸、胜利、技能等全套音效，体验更沉浸。
核心玩法
左键点击：翻开格子，若为地雷则游戏结束；若为数字则显示周围地雷数量；若为空白则自动连锁展开。
右键点击：对格子进行插旗 / 取消插旗标记，用于标记疑似地雷位置。
胜利条件：翻开所有安全格子 + 正确标记所有地雷。
失败条件：点击到任意地雷，游戏立即结束并显示所有地雷位置。
特色功能
完整游戏系统
实时计时，记录通关速度
动态显示剩余地雷数量
一键重新开始、返回难度选择菜单
智能连锁展开，操作更高效
五大辅助技能（限次使用）
数字标记：高亮所有安全格子，辅助快速判断
安全探测：随机提示一个安全位置
排除地雷：直接标记一个安全格
区域扫描：点击格子，扫描周围 8 格风险
自动插旗：随机为一个未标记的地雷自动插旗
视听体验优化
柔和清爽的 UI 配色，视觉舒适
完整音效系统：背景音乐、点击、爆炸、胜利、技能音效
状态表情实时变化（游戏中 / 失败 / 胜利）
